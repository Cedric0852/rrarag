# RRA Chatbot v1.0

## New Features
- Multi-language support with Google Cloud Translation
- RAG implementation with Qdrant
- Groq LLM integration
- Streamlit UI with chat interface

## Dependencies
- Python 3.10+
- Required API Keys:
  - Google Cloud Translation API
  - Groq API
  - Cohere API
  - Qdrant API

## Installation
```bash
pip install -r requirements.txt
```

## Configuration
1. Set up secrets.toml with required API keys
2. Configure Qdrant database connection
3. Set up Google Cloud credentials
